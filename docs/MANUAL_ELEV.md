# Elevmanual

Den här manualen visar hur eleven använder appen.

## 1. Logga in

1. Skriv in inloggningsnamn.
2. Skriv lösenord.
3. Klicka `Logga in`.

Tips:
- Inloggningsnamn normaliseras (stora/små bokstäver spelar mindre roll).
- Om klassen skapats via lärarens klasslista är startlösenord ofta elevens namn.

## 2. Startsidan

På elevens startsida finns normalt:
- knapp för att starta pass (uppdrag/fri träning),
- tabellövning (2-12),
- val av räknesätt,
- sektionen `Framsteg`,
- eventuellt `Aktiv ticket`,
- byte av elevlösenord.

## 3. Välj träning

Elev kan starta:
1. Fri träning.
2. Ett specifikt räknesätt.
3. Tabellövning (val av tabeller).
4. Nivåfokus via `Framsteg` (klicka en nivå-ruta).

Tempo (challenge/steady) kan väljas på startsidan.

## 4. Under en uppgift

- Uppgift visas med stor text.
- Inmatning kan göras med tangentbord eller knappsats på skärmen.
- Rityta kan visas/döljas under uppgiften.

Knappar:
- `Svara` när eleven svarar på uppgift.
- `Nästa` efter feedback.
- `Radera` tar bort sista tecknet.
- `Rensa` tömmer hela svaret.

## 5. Feedback

- Vid rätt svar visas korrekt feedback och appen kan gå vidare automatiskt.
- Vid fel svar visas `Du svarade ...` i rött och eleven går vidare när hen är redo.

## 6. Pausförslag

Appen kan föreslå kort paus efter en mängd uppgifter.
Eleven kan fortsätta direkt (ingen hård låsning).
Om eleven vill spela en snabb pausaktivitet finns:
- `Pong` (max 2 minuter),
- `Snake` (max 2 minuter).

## 7. Ticket

Om läraren skickat ticket kan eleven se den:
1. via direktlänk,
2. eller tydligt på startsidan som `Aktiv ticket`.

Eleven svarar i ticketvyn och kan beroende på lärarens inställning:
- se rätt/fel direkt,
- eller få facit/rättning senare.

## 8. Framsteg

I `Framsteg` ser eleven nivå-rutor per räknesätt.
Farger:
- Grön: klarad nivå.
- Blå: påbörjad.
- Grå: ej startad.

Elev kan klicka nivå-ruta för att träna just den nivån.

## 9. Avsluta pass

Eleven kan alltid gå tillbaka till sin startsida utan att logga ut och in igen.

## 10. Byta elevlösenord

I startsidan finns sektionen `Byt elevlösenord`.
Eleven anger nuvarande lösenord och nytt lösenord.
